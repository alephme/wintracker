# WindowHack
一个用于查看当前活动窗口信息的 Windows 工具，支持右键菜单。

## 功能
- 最初用来追踪发起弹窗的程序
- 显示活动窗口标题、进程名、PID、程序路径、窗口句柄
- 置顶显示但不影响其他窗口焦点
- Windows 11 右键菜单入口（支持桌面/文件/文件夹/背景）
